Bézout's Identity

∃ integers x,y: ax + by = gcd(a,b)

Solutions not unique. Add k·b to x, subtract k·a from y.

ax+by=c has int. solutions ⟺ gcd(a,b)|c Finding a⁻¹ mod M: solve ax+My=1 via EEA → a⁻¹ = x mod M
Extended Euclidean Algorithm

Returns g, x, y s.t. g = gcd(a,b) and ax+by = g

r₀=a, r₁=b; s₀=1,s₁=0; t₀=0,t₁=1 while rᵢ > 0: qᵢ = ⌊rᵢ₋₁/rᵢ⌋ rᵢ₊₁ = rᵢ₋₁ − qᵢrᵢ sᵢ₊₁ = sᵢ₋₁ − qᵢsᵢ tᵢ₊₁ = tᵢ₋₁ − qᵢtᵢ return rᵢ₋₁, sᵢ₋₁, tᵢ₋₁

Complexity: O(log²a) · runs O(log b) iterations

Two Key Uses of EEA

1. Solve ax+by=c when gcd(a,b)|c → scale Bézout solution by c/gcd

2. Find a⁻¹ mod M: run EEA on (a,M), get ax+My=1 → a⁻¹ = x mod M

ax≡1(mod M) solvable iff gcd(a,M)=1, i.e. a ∈ Z*_M

Multiplicative Group Z*ₘ
Z*ₘ = {a ∈ Zₘ | gcd(a,M) = 1}

Binary op: multiplication mod M

Identity = 1. Inverse of a: use Bézout → ax ≡ 1 (mod M), so a⁻¹ = x mod M

φ(M) = |Z*ₘ| = # integers in [1,M] coprime to M

M prime → φ(M) = M−1 M = pq → φ(M) = (p−1)(q−1) Sum rule: Σ_{d|M} φ(d) = M

Z*ₚ Cyclic — Proof Sketch

Let N_p(d) = #{a ∈ Z*ₚ : ord_p(a) = d}

Step 1: N_p(d) ∈ {0, φ(d)} for all d | p−1 (if ∃ one element of order d, the d roots of xᵈ≡1 are exactly {a,a²,…,aᵈ} and φ(d) of them have order exactly d) Step 2: Σ_{d|p-1} N_p(d) = p−1 (every element has some order dividing p−1) Step 3: Σ_{d|p-1} φ(d) = p−1 [sum rule] Steps 1+2+3 → N_p(d) = φ(d) for ALL d → N_p(p−1) = φ(p−1) ≥ 1 ✓ generator exists

Bernstein-Vazirani

Task: f(x) = x·s, find secret s. Quantum uses 1 query vs classical n.

Same circuit as DJ: apply Hⁿ → Qf → Hⁿ. Measuring gives |s⟩ with certainty.

DFT vs QFT

DFT = classical matrix on a vector. QFT = same transform on quantum amplitudes. Same formula, different object.

You cannot read all QFT output coefficients — measuring collapses to one sample.

Character examples
HWT (n=3): ε₁₀₁(110) = (−1)^(101·110) 101·110 = 1·1+0·1+1·0 = 1 → (−1)¹ = −1 QFT (N=8): ε₃(5) = e^(2πi·3·5/8) = e^(30πi/8) = e^(7πi/4) = cos(7π/4)+i·sin(7π/4) = (1−i)/√2

HWT: dot product mod 2 → ±1. QFT: multiply then exponentiate → complex phase.

Simon's Problem over Zⁿ₂
Problem

Given f:{0,1}ⁿ→{0,1}ⁿ with f(x)=f(y) ⟺ x=y⊕s. Find s.

Hidden period over group Z₂ⁿ (XOR). Hidden subgroup {0ⁿ,s} ≤ Z₂ⁿ. Two inputs equivalent iff same coset. s=0ⁿ: injective; else every output has exactly two preimages x and x⊕s.

Classical randomized: Ω(2^(n/2)) queries (birthday paradox).

Quantum Circuit
|0ⁿ⟩ ─Hⁿ──────Hⁿ─ measure → yᵢ Of |0ⁿ⟩ ─────────── (ancilla)

After circuit:

p(y) = 1/2^(n−1) if y·s = 0 (mod 2) p(y) = 0 if y·s = 1 (mod 2)
State walkthrough
|0ⁿ⟩|0ⁿ⟩ →[Hⁿ]→ (1/√2ⁿ) Σ_x |x⟩|0ⁿ⟩ →[Of]→ (1/√2ⁿ) Σ_x |x⟩|f(x)⟩ (entangled) →[meas. 2nd]→ (1/√2)(|x₀⟩ + |x₀⊕s⟩) ↑ pair state: two branches differ by exactly s →[Hⁿ]→ ampl. of |y⟩ ∝ (−1)^(x₀·y)·(1+(−1)^(s·y)) s·y=0 → factor=2 (amplitudes reinforce) s·y=1 → factor=0 (destructive interference)

x₀ unknown/random — doesn't matter. Only the interference pattern encodes s.

Classical Postprocessing

Repeat n−1 times to get n−1 lin. indep. yᵢ. Gaussian elimination → solutions {0ⁿ, s'}. Check f(0ⁿ) vs f(s') to identify s.

P(n−1 LI strings) > 1/2 After k runs: P(fail) < e^(−k/2)
Worked: n=2, s=11 (full states)
Pairs: {00↔11, 01↔10}, f(00)=f(11)=a, f(01)=f(10)=b After H⊗2 + oracle: ½((|00⟩+|11⟩)|a⟩ + (|01⟩+|10⟩)|b⟩) Measure 2nd reg → say get |a⟩. 1st reg collapses: (|00⟩+|11⟩)/√2 Apply H⊗2: H⊗2|00⟩ = ½(|00⟩+|01⟩+|10⟩+|11⟩) H⊗2|11⟩ = ½(|00⟩−|01⟩−|10⟩+|11⟩) Sum: ½(|00⟩+|11⟩) ← |01⟩,|10⟩ cancel! Measure → {00, 11} each with prob ½. y·s=0: (y=00) 0+0=0 ✓ (y=11) 1+1=2≡0 ✓ → s₁+s₂=0 → s₁=s₂ → s=11 (non-zero) ✓

⚠ The pair state (|00⟩+|11⟩)/√2 is fixed by H⊗2 because it is an eigenstate (eigenvalue +1) — NOT because H²=I. (H²=I means applying twice returns to start; here applying once already stays.)

Example: n=3, s=101
Constraint: y·s = y₁+y₃ = 0 (mod 2) → y₁=y₃ Valid outputs: {000, 010, 101, 111} Collect 2 LI equations, Gaussian elim → s=101 Verify: f(000) = f(101) ✓

Phase Kickback
Setup

Ancilla qubit in |−⟩ = (|0⟩−|1⟩)/√2. Oracle Q_f acts as:

Q_f|x⟩|y⟩ = |x⟩|y ⊕ f(x)⟩ Q_f|x⟩|−⟩ = (−1)^f(x) |x⟩|−⟩

Phase "kicks back" onto control register. Ancilla unchanged.

Two-case derivation
Case f(x)=0: Q_f|x⟩|−⟩ = |x⟩|0⊕0⟩−|x⟩|1⊕0⟩ / √2 = |x⟩(|0⟩−|1⟩)/√2 = |x⟩|−⟩ = (−1)⁰|x⟩|−⟩ ✓ Case f(x)=1: Q_f|x⟩|−⟩ = |x⟩|0⊕1⟩−|x⟩|1⊕1⟩ / √2 = |x⟩(|1⟩−|0⟩)/√2 = −|x⟩|−⟩ = (−1)¹|x⟩|−⟩ ✓
Oracle unitarity

Q_f is self-inverse: Q_f² = I.

Q_f|x,y⟩ = |x, y⊕f(x)⟩ (XOR is its own inverse) Q_f(Q_f|x,y⟩) = |x, y⊕f(x)⊕f(x)⟩ = |x,y⟩ → Q_f is unitary (bijection on basis states)
Oracle types (distinct!)
Standard oracle: Uf|x⟩|y⟩ = |x⟩|y⊕f(x)⟩ Phase oracle: Of|x⟩ = (−1)^f(x)|x⟩ ↑ no ancilla; f(x) written into sign directly

Phase kickback converts standard oracle + |−⟩ ancilla into phase oracle behaviour.

Why it works

|−⟩ is eigenstate of X with eigenvalue −1. Flipping the ancilla when f(x)=1 multiplies state by −1.

|−⟩ = (|0⟩−|1⟩)/√2 X|−⟩ = −|−⟩ → eigenvalue −1
General eigenstate form
If U|u⟩ = e^(iφ)|u⟩ (eigenstate), then: (|0⟩+|1⟩)/√2 ⊗ |u⟩ →[ctrl-U]→ (|0⟩+e^(iφ)|1⟩)/√2 ⊗ |u⟩ Phase e^(iφ) from target kicks back to control.

Boolean case: U=X, |u⟩=|−⟩, e^(iφ)=−1 → the standard formula.

Preparing |−⟩
|0⟩ →[X]→ |1⟩ →[H]→ |−⟩
Uses

Deutsch, DJ, BV, Grover oracle, conditional phase shifts (2|0⟩⟨0|−I)

Hadamard-Walsh Transform
Single qubit
H = (1/√2)[[1,1],[1,−1]] H|0⟩ = |+⟩, H|1⟩ = |−⟩ H|+⟩ = |0⟩, H|−⟩ = |1⟩ H² = I (self-inverse)
n-qubit HWT
H^⊗n|x⟩ = (1/√2ⁿ) Σ_y (−1)^(x·y) |y⟩ H^⊗n|0ⁿ⟩ = (1/√2ⁿ) Σ_x |x⟩ (uniform superpos.)
Key Properties
H^⊗n is unitary and self-inverse: (H^⊗n)² = I Group: Zⁿ₂ (bitwise XOR) Character: εₛ(x) = (−1)^(s·x)

n-qubit HWT = Fourier transform over Zⁿ₂. Applying H⊗n twice returns to original state.

Worked example: H⊗²|10⟩
H⊗²|10⟩ = H|1⟩ ⊗ H|0⟩ = |−⟩ ⊗ |+⟩ = (1/√2)(|0⟩−|1⟩) ⊗ (1/√2)(|0⟩+|1⟩) = (1/2)(|00⟩+|01⟩−|10⟩−|11⟩) Shortcut: coeff of |yz⟩ = (−1)^(x·y) / √2ⁿ x=10, y=10: (−1)^(1·1+0·0) = −1 ✓ x=10, y=01: (−1)^(1·0+0·1) = +1 ✓

Dot product trick: amp(y) = (−1)^(x·y)/√2ⁿ where x·y = Σᵢ xᵢyᵢ mod 2

Complexity

O(n) = O(log N) gates — one H per qubit in parallel.

Grover's Algorithm
Problem

Search for M marked items among N. f:[N]→{0,1}, f(x)=1 iff x is marked.

Classical: Ω(N/M) queries. Grover: O(√(N/M)) → quadratic speedup.

Circuit
|0ⁿ⟩ ─Hⁿ─[G]─[G]─···─[G]─ measure |0⟩ ─X──H─────────────── |−⟩ (ancilla)

Grover operator G = H⊗ⁿ (2|0ⁿ⟩⟨0ⁿ|−I) H⊗ⁿ · Of

Equivalently: G = (2|ψ⟩⟨ψ|−I) · Of where |ψ⟩ = H^⊗n|0ⁿ⟩ = uniform superpos.
Algorithm Steps

1. Init |0ⁿ⟩|0⟩, apply H⊗ⁿ⊗HX → |ψ⟩|−⟩

2. Apply G exactly k = ⌊(π/4)√(N/M)⌋ times

3. Measure n work qubits → solution w.h.p.

Define subspaces
A = {x : f(x)=0}, B = {x : f(x)=1} |α⟩ = (1/√(N−M)) Σ_{x∈A} |x⟩ |β⟩ = (1/√M) Σ_{x∈B} |x⟩ |ψ⟩ = √((N−M)/N)|α⟩ + √(M/N)|β⟩

⟨α|β⟩ = 0 · G acts in the 2D real subspace spanned by |α⟩,|β⟩.

Grover Analysis — Geometry
Two reflections = rotation

Of = reflection about |α⟩ axis (flips sign of |β⟩ component)

Of(a|α⟩+b|β⟩) = a|α⟩ − b|β⟩ = (2|α⟩⟨α|−I)(a|α⟩+b|β⟩)

(2|ψ⟩⟨ψ|−I) = reflection about |ψ⟩ axis

Composition of two reflections in ℝ² = rotation.

G as rotation matrix

Define angle θ via:

cos(θ/2) = √((N−M)/N) = a sin(θ/2) = √(M/N) = b G = [[cosθ, −sinθ], [sinθ, cosθ]] (in |α⟩,|β⟩ basis) G|α⟩ = cosθ|α⟩ + sinθ|β⟩ G|β⟩ = −sinθ|α⟩ + cosθ|β⟩
After k iterations
Gᵏ|ψ⟩ = cos((2k+1)θ/2)|α⟩ + sin((2k+1)θ/2)|β⟩ P(solution) = sin²((2k+1)θ/2)

Maximise by setting (2k+1)θ/2 ≈ π/2:

k = ⌊π/(2θ) − 1/2⌋ Error ≤ θ (at most one extra rotation)
Iteration count
M ≤ N/2: θ/2 ≥ sin(θ/2) = √(M/N) k ≤ π/(2θ) ≤ (π/4)√(N/M) → k = O(√(N/M)) iterations

If M ≥ N/2: double N with ancilla qubit, mark only when ancilla=|0⟩.

Amplitude amplified: sin²(θ/2) → sin²((2k+1)θ/2) ≈ 1

Shor's Algorithm (Factoring)
Problem

Given N=pq (product of two primes), find p and q. Classical best: sub-exponential. Shor: polynomial O(log³N).

Reduction: Factoring → Period Finding

Pick random a with gcd(a,N)=1. Define:

f(x) = aˣ mod N (periodic with period r) aʳ ≡ 1 (mod N) ← r = ord_N(a)

If r is even and aʳ/² ≢ −1 (mod N):

aʳ − 1 ≡ 0 (mod N) → (aʳ/²−1)(aʳ/²+1) ≡ 0 (mod N) → gcd(aʳ/²±1, N) gives non-trivial factor

Works with prob. ≥ 1/2 over random a. Repeat if r odd or aʳ/²≡−1.

Quantum Period Finding (core)
|0ⁿ⟩ ─QFT──────────QFT⁻¹─ measure → r̃ |0ⁿ⟩ ──────U_f─────────── (f register)
1. Init |0ⁿ⟩|0ⁿ⟩, apply QFT to 1st register 2. Apply U_f: |x⟩|0⟩ → |x⟩|aˣ mod N⟩ 3. Measure 2nd register (collapses 1st to uniform superpos. over {x₀, x₀+r, …}) 4. Apply QFT⁻¹ to 1st register 5. Measure → multiple of N/r 6. Use continued fractions to extract r
Number theory prerequisites
gcd(a,N)=1 ↔ a ∈ Z*_N r = ord_N(a) r | φ(N) = (p−1)(q−1) φ(N) = (p−1)(q−1) for N=pq aᵠ⁽ᴺ⁾ ≡ 1 (mod N) [Euler's theorem]
Why QFT extracts period

After measuring f-register, 1st register is:

|φ⟩ ∝ Σ_{j} |x₀ + jr⟩ (uniform over coset) QFT → peaks at multiples of N/r Continued fractions: s/N ≈ k/r → find r

Modular Arithmetic — The Trick
Working with congruences

To verify a ≡ a' (mod M): write a = kM + a', take mod M on both sides → a ≡ a'.

Given: a ≡ a' (mod M), b ≡ b' (mod M) Write: a = kM + a', b = k'M + b' ab = (kM+a')(k'M+b') = M(kk'M+kb'+k'a') + a'b' → ab ≡ a'b' (mod M) ✓
EEA Use Cases
Use Case 1: Solve ax + by = c

When c = k · gcd(a,b) for some integer k. Run EEA to get x₀, y₀ s.t. ax₀+by₀=gcd(a,b), then scale by k.

General solution: x = kx₀ + (b/d)·t y = ky₀ − (a/d)·t for any integer t, d = gcd(a,b)
Use Case 2: Find multiplicative inverse

Find x s.t. ax ≡ 1 (mod b). Requires gcd(a,b)=1.

Run EEA on (a, b) → get x, y s.t. ax + by = 1 Take mod b: ax ≡ 1 (mod b) → x mod b is the inverse of a

This is how Z*ₘ inverses are computed in practice.

Z*ₘ Inverse — Proof via Bézout

For any a ∈ Z*ₘ, gcd(a,M)=1. Apply Bézout:

∃ x,y ∈ ℤ: ax + My = 1 Take (mod M): ax ≡ 1 (mod M) → (x mod M) is a's inverse in Z*ₘ

Check x mod M also satisfies Bézout: replace x with x mod M = x − kM, then a(x−kM) + M(y+ka) = 1 ✓

Example: Z*₁₅
Z*₁₅ = {1,2,4,7,8,11,13,14} 2·8 = 16 ≡ 1 (mod 15) → 2⁻¹ = 8 7·13 = 91 ≡ 1 (mod 15) → 7⁻¹ = 13
Euler's Theorem — Proof Sketch
Setup

Let Z*ₘ = {x₁, x₂, …, xₖ} where k = φ(M). Since gcd(a,M)=1, a ∈ Z*ₘ.

Key idea: multiplication by a is a permutation
Consider: a·Z*ₘ = {ax₁, ax₂, …, axₖ} Claim: this is the same set as Z*ₘ (just reordered). Why? Because axᵢ ≡ axⱼ (mod M) iff xᵢ = xⱼ (a has an inverse → cancel a from both sides). So all axᵢ are distinct mod M → same set.
Finish
Product of {x₁,…,xₖ} ≡ product of {ax₁,…,axₖ} ∏ xᵢ ≡ aᵏ · ∏ xᵢ (mod M) Since ∏ xᵢ ∈ Z*ₘ → it has an inverse. Multiply both sides by (∏ xᵢ)⁻¹: 1 ≡ aᵏ ≡ aᵠ⁽ᴹ⁾ (mod M) ✓
Order Divides φ(M) — Proof Sketch
Goal:

Show r = ord_M(a) divides φ(M).

Division algorithm trick
By Euler: aᵠ⁽ᴹ⁾ ≡ 1 (mod M) Write φ(M) = q·r + s, 0 ≤ s < r Then: aᵠ⁽ᴹ⁾ = a^(qr+s) = (aʳ)^q · aˢ ≡ 1^q · aˢ ≡ aˢ ≡ 1 (mod M) But r is the SMALLEST positive integer with aʳ ≡ 1. Since 0 ≤ s < r: s cannot be positive → s = 0 → r | φ(M) ✓

Corollary: if p prime, ord_p(a) | p−1 since φ(p)=p−1.

Implication for Shor: the period r of aˣ mod N divides φ(N)=(p−1)(q−1).

Z*ₚ is Cyclic — Proof Sketch
Goal

Show ∃ a ∈ Z*ₚ with ord_p(a) = p−1 (a generator).

Step A — Sum identity
For any M ≥ 1: Σ_{d | M} φ(d) = M Proof: consider fractions 1/M, 2/M, …, M/M. Each reduces to k/d with gcd(k,d)=1. # fractions with denom d = φ(d). Summing over all d|M gives M. ✓
Step B — Nₚ(d) ∈ {0, φ(d)}

Nₚ(d) = #{a ∈ Z*ₚ : ord_p(a) = d}

If Nₚ(d) > 0: ∃ a with ord_p(a)=d. Then {1,a,a²,…,aᵈ⁻¹} are d distinct roots of xᵈ−1 ≡ 0 (mod p). A degree-d poly over a prime field has at most d roots → these are ALL roots. Any b with bᵈ≡1 must be some aᵏ. ord(aᵏ) = d/gcd(k,d) = d iff gcd(k,d)=1 # such k in [0,d−1] = φ(d). → Nₚ(d) = φ(d). ✓
Step C — Conclusion
From B: Nₚ(d) ≤ φ(d) for all d | p−1 Claim: Σ_{d | p−1} Nₚ(d) = p−1 (each element has exactly one order, and every order divides p−1) From A: Σ_{d | p−1} φ(d) = p−1 So both sides equal p−1, and no term exceeds → Nₚ(d) = φ(d) for ALL d. For d = p−1: Nₚ(p−1) = φ(p−1) ≥ 1 → ∃ generator of Z*ₚ ✓
φ(pq) = (p−1)(q−1) — Proof

Count integers in {1,…,pq−1} NOT coprime to pq. Since p,q prime, non-coprime = multiples of p or q.

Multiples of p in range: p, 2p, …, qp → q total Multiples of q in range: q, 2q, …, pq → p total But pq counted twice → subtract 1 Not coprime: q + p − 1 φ(pq) = pq − (p+q−1) = (p−1)(q−1) ✓
Shor — Number Theory Connection
Why period finding works
f(x) = aˣ mod N is periodic with period r r = ord_N(a), and r | φ(N) = (p−1)(q−1) aʳ ≡ 1 (mod N) → (aʳ/²)² ≡ 1 (mod N) → (aʳ/²−1)(aʳ/²+1) ≡ 0 (mod N=pq) N doesn't divide either factor alone (if r even, aʳ/² ≢ ±1) → gcd splits N: gcd(aʳ/²−1, N) = p or q ✓
When does this fail?
Fails if: r is odd, OR aʳ/² ≡ −1 (mod N) Fix: pick a new random a and retry. P(success per attempt) ≥ 1/2.
Common Exam Pitfalls

⚠ Bézout coefficients not unique — EEA and direct solving give different (x,y). Both valid.

⚠ Z*ₘ ≠ Zₘ — Only elements coprime to M are in Z*ₘ. E.g. 3 ∉ Z*₁₅.

⚠ ord divides φ, not equals — ord_M(a) | φ(M), but could be a proper divisor.

⚠ QFT ≠ reading amplitudes — Output amplitudes can't be fully extracted without exponentially many measurements. Use QFT only as intermediate step.

⚠ Grover error ≤ θ — Rounding k to nearest integer means error in angle ≤ θ. Smaller M/N → smaller θ → better precision.

⚠ Simon ZN: L must divide N — Simplification for clean geometric series evaluation.

⚠ Simon pair state: x₀ is random — The specific x₀ doesn't matter; the interference from (|x₀⟩+|x₀⊕s⟩) only depends on s.

Qubit Fundamentals
State & normalization
|ψ⟩ = α|0⟩ + β|1⟩, α,β ∈ ℂ |α|² + |β|² = 1 (probabilities sum to 1) |0⟩ = [1,0]ᵀ, |1⟩ = [0,1]ᵀ |ψ⟩ = [α, β]ᵀ
Measurement (comp. basis)
P(0) = |α|², P(1) = |β|² State collapses to observed basis state. ‼ Amplitudes ≠ probabilities. P = |amp|².
Unitary evolution
U†U = I (norm-preserving, reversible) All quantum gates must be unitary. Classical irreversible AND → embed as: |x,y⟩ → |x, y⊕f(x)⟩
Bra-ket
|ψ⟩ = ket (column), ⟨ψ| = bra (row, conj. T) ⟨φ|ψ⟩ = inner product, ‖ψ‖² = ⟨ψ|ψ⟩ = 1

|+⟩=(|0⟩+|1⟩)/√2, |−⟩=(|0⟩−|1⟩)/√2 H|0⟩=|+⟩, H|1⟩=|−⟩, H²=I Measuring after H = measuring in ±-basis.

n qubits → 2ⁿ-dimensional space (tensor ⊗) (α|0⟩+β|1⟩)⊗(γ|0⟩+δ|1⟩) = αγ|00⟩+αδ|01⟩+βγ|10⟩+βδ|11⟩

No-Cloning Theorem

Statement: No unitary U s.t. for all |ψ⟩:

U(|ψ⟩|0⟩) = |ψ⟩|ψ⟩

Proof (linearity contradiction):

U|0⟩|0⟩=|00⟩, U|1⟩|0⟩=|11⟩ → U(|+⟩|0⟩) = (|00⟩+|11⟩)/√2 but we need |+⟩|+⟩ ∝ |00⟩+|01⟩+|10⟩+|11⟩ → Contradiction. ∎

Classical bits freely copyable; unknown quantum states are not.

Quantum Teleportation

Send unknown |ψ⟩ from Alice → Bob.

Resources: 1 Bell pair + 2 classical bits Steps: 1. Alice does CNOT+H on her qubit+Bell half 2. Alice measures → sends 2 classical bits a,b 3. Bob applies Xᵃ Zᵇ → recovers |ψ⟩

Key points:

• Original |ψ⟩ consumed (no-cloning safe)

• Entanglement is the resource; 2 bits carry corrections

• No faster-than-light info (need classical channel)

Exam Traps
❌ Amplitudes = probabilities
Wrong: P(x) = α(x) Right: P(x) = |α(x)|² Amplitudes are complex; probabilities are real ≥ 0.
❌ Superposition = "both answers"

Superposition alone gives no speedup. The power is interference — bad outcomes cancel, good ones reinforce.

❌ QFT reads out all Fourier coefficients

No. QFT encodes them into amplitudes. Measurement samples one outcome. Full readout would destroy the speedup.

❌ Simon / DJ output the secret directly

No. They output constraints (linear equations) on the secret. Gaussian elimination / gcd finishes the job classically.

❌ Hadamard and QFT are unrelated

Both are Fourier transforms — H⊗ⁿ over (ℤ₂)ⁿ (characters (−1)^(s·x)), QFT over ℤ_N (characters e^(2πisx/N)).

❌ H²=I means every state is fixed by one H
H²=I means applying H twice returns to start. A state fixed by one H is a +1 eigenstate of H. E.g. (|00⟩+|11⟩)/√2 → H⊗2 → same state. NOT all states behave this way (e.g. (|01⟩+|10⟩)/√2 → H⊗2 → (|00⟩−|11⟩)/√2)
