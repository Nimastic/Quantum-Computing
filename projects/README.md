# Deep Projects

These are larger follow-ons to the lecture labs.

- `project_teleportation_diagnostics.py`: inspect all Bell-measurement branches and verify corrections.
- `project_bv_oracle_workbench.py`: sweep many secret strings and study exact recovery in Bernstein-Vazirani.
- `project_simon_hidden_structure.py`: run a larger Simon toy instance with nontrivial classical post-processing.
- `project_shor_factor_15.py`: combine order-finding intuition with classical post-processing to factor 15.

The notebooks under `notebooks/projects/` are generated from these scripts.
